Bienvenido a Neuropulse

Esta plataforma es el centro de análisis de datos para el Guante Inteligente Neuropulse. Diseñado para el monitoreo fisiológico continuo del usuario, recopila métricas clave para evaluar el nivel de estrés y el estado de salud general.

Objetivo del Proyecto:

Neuropulse combina un microcontrolador ESP32 con sensores de Respuesta Galvánica de la Piel (GSR) y un sensor de Pulso para capturar:

  • Nivel de Estrés (GSR): Un indicador objetivo de la actividad del sistema nervioso simpático.
  • Ritmo Cardíaco (BPM): La frecuencia de los latidos del corazón.

Toda la información se transmite vía Wi-Fi para un almacenamiento seguro y análisis detallado en esta interfaz web impulsada por Django y Supabase.

¿Cómo Navegar?

Utilice la barra de navegación superior para acceder a los gráficos interactivos de nivel de estrés (GSR) y ritmo cardíaco, visualizar la correlación entre ambos, o revisar los registros históricos de los usuarios. También puede descargar nuestra aplicación móvil.
